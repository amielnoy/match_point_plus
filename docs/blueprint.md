# **App Name**: MatchSpark

## Core Features:

- User Profile Creation: Users can create profiles with personal information, photos, and interests.
- Profile Matching: The application suggests potential matches based on compatibility algorithms. The LLM can act as a tool by deciding when location or other data are too different for it to suggest someone to the user.
- Direct Messaging: Users can send direct messages to their matches.
- Geolocation Services: Use geolocation to find matches within a certain radius.
- User Preferences: Allow users to set their matching preferences based on age, interests, and location.
- Photo Upload: Enable users to upload and manage profile pictures.

## Style Guidelines:

- Primary color: Vibrant coral (#FF7F50) to evoke feelings of excitement and warmth.
- Background color: Light peach (#FAEBD7), a desaturated tint of coral to maintain a soft and inviting backdrop.
- Accent color: Soft pink (#F08080) complements the coral primary color while providing good contrast for interactive elements.
- Headline font: 'Belleza' sans-serif, suitable for headlines and short chunks of text. Body text: 'Alegreya' serif.
- Use modern, minimalist icons for profile settings, matching, and messaging features.
- Clean and intuitive layout with clear visual hierarchy to ensure ease of navigation.
- Subtle transitions and animations to enhance user engagement and provide feedback on interactions.
